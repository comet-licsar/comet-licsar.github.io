LiCSAR FramebatchÔÉÅ

This is a toolset combining LiCSAR core codes (licsar_proc) with the LiCSInfo database, as described in https://www.mdpi.com/2072-4292/12/15/2430/htm . It contains set of tools to be used by COMET users with access to JASMIN computing facility. The tools help process Sentinel-1 data systematically to generate standard LiCSAR interferometric products.
